## Introdução
"Testes-Integracao" refere-se a um protótipo destinado a testar na prática a utilização, integração e interação de recursos e tecnologias amplamente utilizadas atualmente, com objetivo de encetar o meu 
conhecimento, explorando modelos e conceitos de organização de código, boas práticas, além do ambiente Git/GitHub como GitFlow, GitHub Projects (Kanban), etc.


## Objetivo
Testar na prática conceitos amplamentes utilizados no mercado, com fins de agregar conhecimento para futuramente utilizar em projetos profissionais e pessoais.
Espero aprender com esse projeto, conceitos de: Deploy, variaveis de ambiente (.env), banco de dados em nuvem, conteinerização e escalabilidade.


## Tecnologias e linguagens utilizadas
Banco de dados: MySQL + CleverCloud.
Frontend: React + JavaScript.
Backend: Node.js, TypeORM, Express, Nodemon.
